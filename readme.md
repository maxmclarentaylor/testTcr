# template-tcr-kata
A template repo for trying out any code kata in a `test && commit || revert` workflow, [created by Kent Beck and Oddmund StrÃ¸mme](https://medium.com/@kentbeck_7670/test-commit-revert-870bbd756864).

## steps
1. Write some code, including tests
2. Run `npm run tcr ${your_commit_message}`
3. Think about what you may have learnt
4. Go to step 1
